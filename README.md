# Kopienix
Implementation of linux commands to learn C




## Execution of indiviual commands
gcc <command.c> -o \<command>

eg. 

    gcc date.c -o date
    gcc uname.c -o uname
    gcc pwd.c -o pwd

## Current list of commands
    ./date
    ./uname
    ./pwd
    
uname supports help function (./uname --help) displays list of possible options.
